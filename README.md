# apex-bot

### 指令：
* map
* news
* rank
* server_status
* shop

## map
查詢目前APEX Legends地圖循環

## news
查看APEX Legends新聞

## rank
查看目前APEX Legends牌位狀況，如：PC / PS4 / XBOX / Switch，各裝置頂尖獵殺者數量、"最後一名"頂尖獵殺者分數、大師數量

## server_status
查看目前APEX Legends伺服器狀態

## shop
查看APEX Legends商城
